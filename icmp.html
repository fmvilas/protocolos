    Internet Control Message Protocol

    Esta es la tercera opción del menú (Protocolo ICMP). Desde aquí podremos enviar paquetes ICMP de ECO, MÁSCARA DE SUBRED Y MARCA DE TIEMPO (TIMESTAMP) directamente al destino, al router, o al destino a través del router. Aunque se pueden mandar esos 3 tipos de mensajes ICMP, esta aplicación sólo contesta mensajes de ECO recibidos. Si en el momento de mandar el paquete ICMP no conocemos las direcciones Ethernet necesarias para su envío, el programa automáticamente mandará una solicitud ARP y seguidamente, cuando reciba la respuesta ARP, enviará el paquete ICMP.

    Para más información sobre el protocolo ICMP puedes consultar el RFC 792 en inglés o español.
